### Added
- Initial modular Android SDK (`taskbridge-core`, `taskbridge-transport-okhttp`).
- Reliable task event streaming with WebSocket/SSE/Long-polling.
- Checkpoint-based recovery for Android.
- FastAPI backend implementation.
- Support for JSON and Multipart task creation.
- Task suspension and user action submission.
